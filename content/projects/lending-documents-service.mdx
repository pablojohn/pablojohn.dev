---

Create event driven system using Azure Functions, Azure Service Bus and .NET 6 to take in documents uploaded from web applications and then send/upload them to necessary partners.
System specifically made use of the Pub/Sub pattern via the Azure Service Bus Topics & Subscriptions as well as HTTP Triggered and Service Bus Triggered Azure Functions.
Also integrated with API Gateways like IBM API Connect and Azure API Management to other On-Premise or Cloud based systems.

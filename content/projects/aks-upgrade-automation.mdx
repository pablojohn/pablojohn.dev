Create YAML DevOps pipelines, Powershell scripts and Pester tests to automate migrating applications from an AKS cluster to a new AKS cluster. Things automated include creating, configuring and testing the AKS cluster. Creating, configuring and testing other Azure infrastructure. As well as deploying apps to the new cluster and testing the app on the new cluster.

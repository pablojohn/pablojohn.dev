Design and create new Azure native platform to host large microservice applications that were previously hosted on Pivotal Cloud Foundry.
Azure platform consisted of App Services, App Gateways, Traffic Manager, Azure Kubernetes Service and more, all of which was infrastructure as code via YAML pipelines and deployed with AzureDevOps.
Along with the infrastructure, modernized the microservices which were .NET Framework & .NET Core.
